## Usage
1. Create new decision from `DECISION-TEMPLATES/decision-record.template.md`
2. Create matching held-out scenarios from `DECISION-TEMPLATES/scenario-pack.template.md`
3. Run review using `DECISION-TEMPLATES/review-report.template.md`
4. Update this index with status + pass rate
